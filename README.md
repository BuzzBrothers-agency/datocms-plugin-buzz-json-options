# DatoCMS Plugin Buzz Text Styles

This plugin allows you to define and apply custom text styles to your DatoCMS content fields, enhancing the visual consistency and branding of your content.

- [DatoCMS Plugin Buzz Text Styles](#datocms-plugin-buzz-text-styles)
  - [Features](#features)
  - [Usage](#usage)

![DatoCMS Plugin Buzz Link Preview](./docs/datocms-plugin-buzz-text-styles-preview.webp)

## Features

1. Support all features of the [Structured Text customizations
   ](https://www.datocms.com/docs/plugin-sdk/structured-text-customizations)

## Usage

To create a media grid, simply follow these steps:

1. Go to the plugin configuration page
2. Specify in `JSON` format the styles you want to add by following the [structured text customizations documentation
   ](https://www.datocms.com/docs/plugin-sdk/structured-text-customizations)
